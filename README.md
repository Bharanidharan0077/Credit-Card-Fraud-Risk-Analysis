1️⃣ Project Title
# Credit Card Fraud Risk Analysis

2️⃣ Project Overview
## Project Overview
This project analyzes credit card transaction data to identify fraudulent activities and risk patterns. 
Using Power BI, the dashboard visualizes fraud trends, transaction types, and high-risk categories to help financial institutions detect fraud effectively.

3️⃣ Objectives
## Objectives
- Identify the most common types of fraudulent transactions
- Analyze fraud distribution across different transaction categories
- Detect patterns in fraudulent behavior
- Create an interactive dashboard for fraud monitoring

4️⃣ Dataset Information
## Dataset
- Source: Credit Card Transaction Dataset
- Records: 8000+ transactions
- Features:
  - Transaction ID
  - Transaction Type
  - Amount
  - Location
  - Fraud Flag
